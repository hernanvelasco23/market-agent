# market-agent
AI-powered market monitoring system built with .NET, Semantic Kernel, and Azure OpenAI.

I wanted a project that combined real backend engineering with practical agentic AI. So I designed Market Agent as a .NET application that ingests market data, detects transparent rule-based signals, and uses Semantic Kernel with Azure OpenAI to generate structured daily briefings. A key design decision was keeping market logic deterministic and using the LLM only for synthesis and interpretation. I’m also building it with proposal, design, and task documents per feature to keep the workflow agent-friendly and auditable.
